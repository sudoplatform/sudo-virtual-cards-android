provisionCard

 abstract suspend fun provisionCard(input: ProvisionCardInput): ProvisionalCard

Provision a virtual payment card. Initially a ProvisionalCard is returned with the details of the card being created. You should query the state of this card periodically by calling getProvisionalCard to monitor the card's progress from a state of ProvisionalCard.State.PROVISIONING to ProvisionalCard.State.COMPLETED or ProvisionalCard.State.FAILED.

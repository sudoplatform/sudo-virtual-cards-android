abstract suspend fun provisionVirtualCard(input: ProvisionVirtualCardInput): ProvisionalVirtualCard

Provision a VirtualCard.

Initially a ProvisionalVirtualCard is returned with the details of the virtual card being created. The state of this virtual card can be queried periodically by calling getProvisionalCard to monitor the card's progress from a provisioning state to a completed or failed state.

Return

A ProvisionalVirtualCard in a provisioning state.

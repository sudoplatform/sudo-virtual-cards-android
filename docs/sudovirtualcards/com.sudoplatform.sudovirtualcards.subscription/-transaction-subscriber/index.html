Notifies the subscriber that the subscription connection state has changed. The subscriber won't be notified of Transaction changes until the connection status changes to ConnectionState.CONNECTED. The subscriber will stop receiving Transaction change notifications when the connection state changes to ConnectionState.DISCONNECTED.
